# 接口
## 定义
1. 集合了一组方法（0~3），但是没有具体的实现代码
2. 命名一般是方法名后加er，或able
3. 定义格式
   ```go
   type Namer interface{
   Method1(参数)返回值
   }
   ```
4. 干什么？
	1. "如果一只鸟啥都像鸭子，它就是鸭子"，指的是，实现了接口所有方法的类型，一个接口类型可以持有任意其中之一，但不能同时持有两种类型
	2. 每个新类型要实现接口，就必须自己写一遍接口要求的方法，但是使用接口类型编写的代码，可以接受任何实现了该接口的新类型，不需要修改
5. 注意：
	   1. 实现了接口方法的是a类型的指针变量而不是该类型时，接口可选的动态类型是指针a
	   2. 实现了接口时a类型时，a和a的指针都可以
## 怎么用
1. 调用接口方法的函数
	1. 与定义方法不同，接口需要作为接口调用函数的形式参数
	2. 然后按照变量名.方法名使用即可
2. 接口嵌套
   嵌套了接口的接口可以使用子接口的所有方法
