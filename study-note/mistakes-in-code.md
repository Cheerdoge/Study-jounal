# 语法
1. 有返回值的函数忘写return
2. 用make来新建结构体变量
3. copy到一个长度为0的切片，没有元素被复制
4. 创建一个空切片忘了`make（type, 0, cap）`
5. 输入切片容量和元素没有对输入容量时的\n进行处理，导致录入元素时，最后一个没有被录入
6. 一次递归指向多个二次递归时直接在函数体内对全局变量变化，导致混乱，应在函数参数中变化
7. 在函数体内直接新声明一个函数
8. 在函数体外使用`:=`
9. 仅仅用`var xxx map[type]type`声明map而不初始化，应用`var xxx = make(map[type]type)`
10. Header是一个字符串切片的字典，却直接打印
11. 把Header当做和Body一样的类型，放入io.ReaderAll读取
