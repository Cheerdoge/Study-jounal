发现不咋用函数，结果现在只会最基本的，匿名函数啥的更是忘得差不多了，麻了麻了，有点忘了的就记下来了，开学！
1. 引用其他包里的函数格式需要加函数名`fmt.Println`，主要是自定义一个包时别忘了
2. `type aaa func(int) int`如此将声明一个函数类型，不需要函数体，可以赋给变量
3. 值传递不改变原值，引用改变原值，传入参数时用&即可（指针），而且消耗比值传递少，还不需要`return` 返回
	但是**切片（slice）、字典（map）、接口（interface）、通道（channel）** 默认引用
4. 有多个`defer` 时，后进先出
5. ```go
   func a() {

    defer un(trace("a"))

    fmt.Println("in a")

}
   ```
   这段代码会先执行trace函数，再最后执行un函数
6. `defer` 工作流程：`return` 后的语句执行-->`defer` 中语句执行-->`return` 
7. 闭包中使用的变量会一直保留到最后一个闭包引用完毕，没有被闭包引用的局部变量会在当前函数返回后被回收
8. **闭包内的变量不会被保留状态** 
9. 学会**避免重复计算** 来优化性能，例如斐波那契数列用闭包而不是递归，因为闭包每次计算都是原本没有的值
